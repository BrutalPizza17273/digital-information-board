# digital-information-board
A fully customisable information board with dynamic colour schemes based on the time of day.
This project is not completed, and is subject to possible changes

- Firstly, I want it to be able to display various pieces of information: Time, Date, Weather, To-do list. Each displayed on their own screens
- The ability to add and group backgrounds based on the time of day (eg. Use backgrounds from group A when its morning, or Group B at night). In the future, I may extend to more conditions (eg. Month, Weather, etc instead of time)
- A tactile way to transition between different screens (ie. Not simply a touchscreen)
- The option to connect it wired or from battery power.
- A magnetic backing for easy adhesion to surfaces.
